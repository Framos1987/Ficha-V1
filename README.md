# Ficha de RPG - Sistema Digital

Bem-vindo ao aplicativo da nossa Ficha de RPG! Este é um sistema projetado para automatizar o gerenciamento de personagens, atributos, inventário, armas e engaste de gemas.

## Recursos
- **Cálculo Automático**: Toda a matemática para combate, perícias, aptidões e derivados é resolvida passivamente.
- **Inventário Dinâmico**: Calcule limite de carga automaticamente.
- **Engaste de Gemas**: Personalize anéis, colares e cintos com pedras preciosas.
- **Design Responsivo**: Use confortavelmente na tela do computador ou do smartphone.

## Como Jogar / Abrir a Ficha
Para os jogadores: basta acessar a URL onde a ficha está hospedada. O navegador lidará com o salvamento local dos seus personagens. Você não precisa configurar nenhum banco de dados ou instalar programas pesados!
